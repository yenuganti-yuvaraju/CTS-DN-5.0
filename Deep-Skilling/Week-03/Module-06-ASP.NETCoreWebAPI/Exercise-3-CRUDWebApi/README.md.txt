# Exercise 3 – CRUD Web API

## Objective
Implement CRUD operations using ASP.NET Core Web API.

## Endpoints

### GET /api/products
Returns all products.

### GET /api/products/{id}
Returns a product by ID.

### POST /api/products
Adds a new product.

### PUT /api/products/{id}
Updates an existing product.

### DELETE /api/products/{id}
Deletes a product.

## Testing
All APIs were successfully tested using Swagger UI.
