---

# 📖 What is Artificial Intelligence (AI)?

Artificial Intelligence (AI) is a branch of computer science that enables machines to perform tasks that normally require human intelligence.

These tasks include:

- Learning
- Reasoning
- Problem Solving
- Decision Making
- Speech Recognition
- Image Recognition
- Natural Language Processing

---

# 🧠 What is Machine Learning (ML)?

Machine Learning is a subset of Artificial Intelligence that enables computers to learn from data without being explicitly programmed.

### Types of Machine Learning

- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning

### Applications

- Spam Detection
- Recommendation Systems
- Fraud Detection
- Medical Diagnosis

---

# 🔬 What is Deep Learning?

Deep Learning is a subset of Machine Learning that uses Artificial Neural Networks with multiple hidden layers to learn complex patterns from large datasets.

### Applications

- Image Recognition
- Speech Recognition
- Self-Driving Cars
- Medical Image Analysis

---

# ✨ What is Generative AI?

Generative Artificial Intelligence (GenAI) is a type of AI that can generate new content based on user input.

It can create:

- Text
- Images
- Videos
- Audio
- Source Code
- Presentations

Unlike traditional AI, which mainly analyzes existing data, Generative AI creates new content.

---

# 📝 Large Language Models (LLMs)

Large Language Models are AI models trained on massive amounts of text data to understand and generate human-like language.

### Popular LLMs

- ChatGPT
- GPT-4
- Gemini
- Claude
- Llama
- Mistral

### Capabilities

- Answer Questions
- Summarize Text
- Translate Languages
- Generate Code
- Create Documents
- Write Emails

---

# 💬 Prompt Engineering

Prompt Engineering is the process of designing effective prompts to obtain accurate and useful responses from Generative AI models.

### Characteristics of a Good Prompt

- Clear
- Specific
- Context-Aware
- Goal-Oriented

### Example

**Prompt**

```
Explain Docker in simple language with examples.
```
