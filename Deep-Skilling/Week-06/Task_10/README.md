# HOL 10 - Office Space Rental App

## Aim
Create a React JSX app to display office rental listings with conditional rent coloring.

## Features
- JSX heading and office SVG image
- Office objects with Name, Rent, Address
- List rendered using `.map()`
- Rent < 60000 shown in red, Rent > 60000 shown in green

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Office cards in a grid layout with rent color-coded by price range.
