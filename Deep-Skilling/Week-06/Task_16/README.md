# HOL 16 - Mail Register App

## Aim
Create a registration form with real-time validation using onChange and onSubmit.

## Features
- `Register` functional component
- Form fields: name, email, password
- Validation:
  - Name: minimum 5 characters
  - Email: must contain `@` and `.`
  - Password: minimum 8 characters
- onChange and onSubmit validation
- Error messages displayed under each field
- Alert on successful registration

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Registration form with live inline validation and success message on valid submission.
