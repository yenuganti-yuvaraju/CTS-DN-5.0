# HOL 18 - Cohort Unit Testing App

## Aim
Write unit tests for the CohortDetails component using Vitest and React Testing Library.

## Features
- `CohortDetails` component with data-testid attributes
- `CohortData.js` with mock data
- `CohortDetails.test.jsx` with tests:
  - should create the component
  - should initialize props
  - should display cohort code in h3
  - should verify other displayed details
  - snapshot test

## How to Run (App)
```bash
npm install
npm run dev
```

## How to Run Tests
```bash
npm install
npm test
```

## Output
All tests pass. The app displays cohort details with selector buttons.
