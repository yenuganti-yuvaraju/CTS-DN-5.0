# HOL 12 - Ticket Booking App

## Aim
Implement conditional rendering to show different views for guest and logged-in users.

## Features
- Guest user: can view flight details only
- Logged-in user: can book tickets
- Login and Logout buttons toggle user state
- GuestPage shows flight list
- UserPage shows flight list with Book button

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Guest sees flights without booking. After login, user can click Book and see confirmation.
