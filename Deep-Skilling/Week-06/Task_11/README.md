# HOL 11 - Event Examples App

## Aim
Demonstrate various React event handling patterns.

## Features
1. Counter with Increment (calls multiple methods) and Decrement buttons
2. Say Welcome button passing "welcome" as argument
3. Synthetic event showing event type and target
4. CurrencyConvertor: Indian Rupees to Euro on form submit

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Four event demo sections on one page: counter, argument passing, synthetic event, currency converter.
