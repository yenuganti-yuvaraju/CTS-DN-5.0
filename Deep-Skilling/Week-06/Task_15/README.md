# HOL 15 - Ticket Raising App

## Aim
Create a complaint registration form that generates a random reference number on submit.

## Features
- `ComplaintRegister` functional component
- Form fields: employee name (textbox), complaint (textarea)
- On submit: generates random reference number (TKT-XXXXXX)
- Alert shown with reference number
- Submitted tickets listed below form

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Form with name and complaint fields. On submit, shows an alert with reference number and adds the ticket to the list.
