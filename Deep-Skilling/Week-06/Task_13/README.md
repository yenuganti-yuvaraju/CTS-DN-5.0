# HOL 13 - Blogger App

## Aim
Demonstrate conditional rendering (multiple ways) and list rendering with keys.

## Features
- `BookDetails`: ternary conditional rendering, list with keys
- `BlogDetails`: if-else in render, `&&` operator rendering (class component)
- `CourseDetails`: switch-based conditional styling, list with keys

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Three detail panels side-by-side with books, blogs, and courses — each showing different conditional rendering approaches.
