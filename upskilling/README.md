* Window Analysis
* Reporting Queries

## Files

```text
SQL_Module
│
├── schema.sql
├── data.sql
└── queries.sql
```

## File Purpose

### schema.sql

Creates:

* Database
* Tables
* Constraints
* Foreign Keys

* Feedback
